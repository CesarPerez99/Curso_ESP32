# Sincronización Avanzada con FreeRTOS en ESP32
## Objetivo de la práctica 
Implementar un sistema completo que demuestre los principales mecanismos de sincronización de FreeRTOS:
- Colas (Queues): Para comunicación entre tareas tipo productor-consumidor
- Semáforos: Para sincronización y control de acceso
- Event Groups: Para coordinación de múltiples eventos
- Mutex: Para proteger recursos compartidos
